Bug Race

Racing game written in ClojureScript

I recently found a repo filled with fun project ideas, and decided to try out one of the advanced projects. Using Re-frame, I was able to finish it in half a Saturday. Re-frame's functional reactive programming architecture makes state management so easy that you barely have to think about it. The hardest part was getting the layout of the control panel to look halfway decent! The only JavaScript interop I used was the setInterval function for a ticker to control the bugs. The ticker interval is fixed, and the bug's speed is a number of pixels to move per tick. This allows the bug speed to be adjusted during the race without having to stop and start the ticker.

Monte Hall

Interactive Monty Hall Simulator written in ClojureScript

This is my first attempt at building a webapp with Re-Frame. The trickiest part was figuring out how to properly handle random number generation within the framework. Since random number generation is effectful, you have to register a "coeffect" that defines how to generate the random number when needed. Then the event handler can inject this coeffect into the coeffect map before the handler actually gets called. This allows the event handler itself to remain side-effect free.
